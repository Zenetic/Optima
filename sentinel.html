SENTINEL Modules

Clinical Trial Eligibility Screening

Screens the HFrEF population for eligibility against active clinical trials, continuously and in the background — the same SENTINEL technology described on this page.

Hybrid Randomized Clinical Trial Design Screening

Adapts time-and-motion methodology to evaluate retrospective patient cases against the qualifying features of a trial's control cohort, supporting hybrid randomized trial designs.

Features: patient-personalized, workflow-integrated (triggered when actually needed, not in isolation), immediately actionable, clinically justified. Deployment: FHIR API interoperability, installable on AWS Cloud / other cloud services, HIPAA certified, meets the cybersecurity standards required by Academic Medical Centers.
